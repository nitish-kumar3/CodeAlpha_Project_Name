# Social App Backend

## Setup

1. Install dependencies:
   ```
   npm install
   ```

2. Create a `.env` file using the provided template

3. Start the server:
   ```
   node index.js
   ```

## Deployment

Deploy to [Render](https://render.com) as a Web Service with these ENV vars:

- `MONGO_URI`
- `JWT_SECRET`
- `CLOUDINARY_CLOUD_NAME`
- `CLOUDINARY_API_KEY`
- `CLOUDINARY_API_SECRET`
